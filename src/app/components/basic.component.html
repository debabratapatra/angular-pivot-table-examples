Basic Pivot Table

It is a very basic pivot table. You provide an array of objects and set rows and columns in configs.

HTML

    {{"
    <db-angular-pivot-table #angularGrid [source]='source' [configs]='configs'></db-angular-pivot-table>
    "}}

rows and columns fields are mandatory.

Please check the sample code below.
{{"
export class BasicPivotTableComponent {
  
    source: any = [
      {'author': 'Deba', 'book': 'Angular'},
      {'author': 'Deba', 'book': 'Physics'},
      {'author': 'Aditya', 'book': 'Angular'}
    ];
  
    configs: any = {
      'rows': 'author',
      'columns': 'book'
    };
}
"}}
